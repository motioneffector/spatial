# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-13

### Added
- Initial release
- Spatial graph with directional navigation
- Support for compass directions (N, S, E, W, NE, SE, SW, NW)
- Vertical movement (UP, DOWN) and special directions (IN, OUT)
- Gated connections with locks, keys, and conditions
- Hidden passages and one-way connections
- Pathfinding with Dijkstra's algorithm
- Tile-based positioning for visual mapping
- Event system for graph changes
- Serialization and deserialization
- Zone-based organization
- Custom direction registration
- Full TypeScript support
- Comprehensive test suite
